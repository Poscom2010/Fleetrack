# FleetTrack Favicon Design

## Design Elements

The FleetTrack favicon features a distinctive kangaroo design representing speed, agility, and power in fleet management:

### Visual Components:
1. **Rounded Square Background** - Light gray (#F5F5F5) with coral/salmon border (#C9746C)
2. **Kangaroo Silhouette** - Dynamic jumping pose symbolizing movement and efficiency
3. **Distinctive Features**:
   - Powerful hind legs - Representing strength and capability
   - Long curved tail - Balance and stability
   - Alert ears - Awareness and monitoring
   - Forward-leaning posture - Progress and momentum

### Color Scheme:
- **Primary**: Coral/Salmon red (#C9746C) - Warm, energetic, professional
- **Background**: Light gray (#F5F5F5) - Clean, neutral canvas
- **Border**: Rounded square with matching coral color

### Symbolism:
- **Kangaroo**: Speed, agility, and power - perfect for fleet management
- **Jumping Pose**: Forward momentum, progress, efficiency
- **Strong Legs**: Capability to handle heavy workloads
- **Long Tail**: Stability and balance in operations
- **Rounded Square**: Professional, contained, complete solution

## File Format
- **SVG**: Scalable vector format for crisp display at any size
- **Optimized**: Clean code, small file size
- **Responsive**: Works perfectly on all devices and screen sizes

## Usage
The favicon automatically appears in:
- Browser tabs
- Bookmarks
- Mobile home screen icons
- PWA app icons
- Search results
